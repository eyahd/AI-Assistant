Arrays in JavaScript

Arrays are used to store multiple values in a single variable.

  • Creating an array: let fruits = ["apple", "banana", "cherry"];
  • Accessing items: Use index, e.g., fruits[0]
  • Looping through arrays: Use for or forEach()
  • Array methods: push(), pop(), length, etc.
