CSS Variables

CSS variables (also known as custom properties) let you store reusable values and keep your stylesheets cleaner and easier to maintain.

  • Declaring a variable: --main-color: pink; (inside a selector)
  • Using a variable: color: var(--main-color);
  • Scope: Variables can be global (in :root) or local.
  • Benefits: Easier theme management, consistent styling, clean code.
