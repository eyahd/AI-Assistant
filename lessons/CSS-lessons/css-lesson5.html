Typography in CSS

Typography defines how your text looks and feels. With CSS, you can customize fonts, sizes, spacing, alignment, and more.

  • font-family: Sets the typeface (e.g., Arial, Georgia, Poppins).
  • font-size: Controls the size of text using rem, px, %, etc.
  • font-weight: Determines how bold the text is.
  • line-height: Controls the vertical spacing between lines.
  • letter-spacing: Adjusts spacing between characters.
  • text-align: Aligns text (left, right, center, justify).
