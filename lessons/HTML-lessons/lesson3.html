HTML Text Formatting

HTML provides several tags to format text, helping developers emphasize content, organize information, and style elements effectively. Let's explore some of the most commonly used text formatting tags:

  • <b> and <strong> for bold text
  • <i> and <em> for italic text
  • <u> for underlined text
  • <mark> to highlight
  • <small> to reduce font size
  • <sub> and <sup> for subscript and superscript
  • <br> for line breaks and <hr> for horizontal rules

Example:

<p>This is <strong>bold</strong>, <em>italic</em>, and <mark>highlighted</mark> text.</p>
<p>H<sub>2</sub>O is the chemical formula for water.</p>
<p>2<sup>3</sup> = 8</p>
<hr>
<p>This is a new line.<br>Another line.</p>
