Lesson 6: HTML Tables

Learn how to display and organize tabular data with HTML tables.

Basic Table Structure

Tables are created using <table>, <tr> (table row), <td> (table data), and <th> (table header).

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>24</td>
  </tr>
</table>
