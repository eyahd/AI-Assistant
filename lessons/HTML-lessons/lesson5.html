Lesson 5: HTML Forms

Learn how to collect user input using forms, inputs, buttons, and more.

Creating Forms

Forms are created with the <form> tag and can include various types of input elements.

<form action="/submit" method="post">
  <label>Name: <input type="text" name="name"></label>
  <button type="submit">Send</button>
</form>

Input Types

Inputs can be text fields, checkboxes, radio buttons, etc.

<input type="email" name="email">
<input type="checkbox" name="subscribe"> Subscribe
