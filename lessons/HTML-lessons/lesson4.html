Lesson 4: HTML Links & Images

Learn how to add links and images to make your webpages interactive and beautiful!

Adding Links

In HTML, links are created using the <a> tag. The href attribute defines the URL the link points to.

<a href="https://example.com">Visit Example</a>

Adding Images

To add images, use the <img> tag. Set the src attribute to the image URL and the alt attribute for accessibility.

<img src="image.jpg" alt="Description of image">
